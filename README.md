# Pong Game

Jogo simples de Pong desenvolvido com HTML5, CSS e JavaScript.

## Como jogar

- Use as teclas **W** e **S** para mover a paleta esquerda.
- Use as setas **↑** e **↓** para mover a paleta direita.
- O objetivo do jogo é rebater a bola e marcar pontos. O jogo termina quando a bola sai da tela.
  
## Instalação

Clone o repositório e abra o arquivo  em seu navegador para jogar.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript
